# pybuf
Modularize Protocol Buffer Compiler generated code for use within your package.

### Version
v1.0


### Installation
```sh
pip install pybuf
```


### Quickstart
```python
from pybuf import modularize


modularize(source='/path/to/proto/files/', destination='/path/to/module/')
```
